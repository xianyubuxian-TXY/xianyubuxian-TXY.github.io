# 1.题目描述
![](https://cdn.jsdelivr.net/gh/xianyubuxian-TXY/hexo-images/images/20260315205542997.png)
- 错误想法：右指针不断向右，遇到重复数字，就得一段；然后左指针设置为当前右指针位置，右指针继续向右扩展
![](https://cdn.jsdelivr.net/gh/xianyubuxian-TXY/hexo-images/images/20260315205727131.png)

# 2.解题思路
- 右指针向右扩展，遇到重复数字时，左指针向右收缩，直到重复数据被排除

# 3.代码实现
```cpp
class Solution {
public:
    /**
     * 代码中的类名、方法名、参数名已经指定，请勿修改，直接返回方法规定的值即可
     *
     * 
     * @param arr int整型vector the array
     * @return int整型
     */
    int maxLength(vector<int>& arr) {
        // write code here
        int len=arr.size();
        if(len<=1) return 1;
        int max_len=0;
        unordered_set<int> Set;
        int left=0;     // 左指针
        for(int right=0;right<len;right++){ // 右指针不断向右扩展
            // 出现重复状态
            while(Set.find(arr[right])!=Set.end()){
                // 出现重复数字，left不断向右收缩，同时Set不断清除数据，知道不在满足重复状态
                Set.erase(arr[left++]);
            }
            // 不是重复数字，加入到Set中，并更新max_len
            Set.insert(arr[right]);
            max_len=max(max_len,right-left+1);
        }
        return max_len;
    }
};
```
