# 2.解题思路与代码实现
## （1）左右扫描法
- 先从左到右扫描，记录每个柱子左边的“最高柱子”
- 在从右到左扫描，记录每个柱子右边的“最高柱子”
- 这两个柱子可以构成“容器”壁，它们之间的“矮柱子”则构成“容器底部”
- 如果一个柱子是“容器壁”，则无法容纳雨水；如果一个柱子是“容器底部”，则能够容纳的雨水数量为两个“容器壁”中矮的那个与当前柱子的高度差（木桶效应）
```cpp
#include <climits>
class Solution {
public:
    /**
     * 代码中的类名、方法名、参数名已经指定，请勿修改，直接返回方法规定的值即可
     *
     * max water
     * @param arr int整型vector the array
     * @return long长整型
     */
    long long maxWater(vector<int>& arr) {
        // write code here
        int len=arr.size();
        if(len<=2) return 0;
        int left=0,right=len-1;
        vector<int> nums(len,INT_MAX);
        int max_height=arr[left];   // 从左到右，初始化最高柱子高度

        // 从左到右遍历
        while(left<len-1){
            left++; // 从左到右时，第一根柱子能够接水数量为0
            if(arr[left]>max_height){
                // 更新最高柱子高度
                max_height=arr[left];
                nums[left]=0;
            }else{
                // 计算当前能接的雨水数，取最小值
                nums[left]=min(nums[left],max_height-arr[left]);
            }
        }

        max_height=arr[right];
        // 从右到左遍历
        while(right>0){
            right--;
            if(arr[right]>max_height){
                max_height=arr[right];
                nums[right]=0;
            }else{
                nums[right]=min(nums[right],max_height-arr[right]);
            }
        }

        int res=0;
        for(int i=1;i<len-1;i++){
            if(nums[i]!=INT_MAX) res+=nums[i];
        }
        return res;
    }
};
```

## （2）单调栈法
- 在“（1）左右扫描法”的基础上，可以想到“单调栈法”，但需要想清楚过程
	- 1.维护一个“单减栈”
	- 2.不能每次有元素从栈中弹出就立即计算，因为之后可能会进来更高的，就重复了
	- 3.计算时机如下：
		- <1>如果入栈元素比栈中元素都大，则入栈元素可以与栈底元素构成容器，这是可以开始计算它们之间能够盛放的水
		- <2>当数组中所有元素都遍历后，栈中剩余元素两两之间都构成容器，这时依次计算它们之间可以盛放的水
	- 4.为了方便计算，入栈元素可以设计为“<数组下标，柱子高度>”
```cpp
class Solution {
public:
    /**
     * 代码中的类名、方法名、参数名已经指定，请勿修改，直接返回方法规定的值即可
     *
     * max water
     * @param arr int整型vector the array
     * @return long长整型
     */
    long long maxWater(vector<int>& arr) {
        int len=arr.size();
        if(len<=2) return 0;
        int res=0;
        stack<pair<int,int>> Stack; // 存储{index,height}的单调递减栈
        Stack.push({0,arr[0]});
        for(int i=1;i<len;i++){
            int val=arr[i];
            pair<int,int> tmp_val;    // 记录出栈的元素，如果都出栈，则正好是栈底元素 
            // 维持单调递减栈
            while(!Stack.empty()){
                tmp_val=Stack.top();
                if(val>tmp_val.second){
                    Stack.pop();
                }else{
                    // 当前元素入栈
                    Stack.push({i,val});
                    break;
                }
            }

            // 栈空：都出栈了，而“入栈元素”与“最后出栈元素”可以形成“容器”盛水
            if(Stack.empty()){
                // 计算它们之间可以陈放的水
                for(int j=tmp_val.first+1;j<i;j++){
                    res+=tmp_val.second-arr[j]; 
                }
                
                // 当前元素入栈
                Stack.push({i,val});
            }
        }

        // 栈中没有剩余元素，则直接结束
        if(Stack.empty()) return res;

        // 遍历完arr数组后，栈中剩余元素之间两两组成容器，依次计算即可
        pair<int,int> tmp_val=Stack.top();  // 记录出栈元素
        Stack.pop();
        while(!Stack.empty()){
            for(int i=Stack.top().first+1;i<tmp_val.first;i++){
                res+=tmp_val.second-arr[i];
            }
            // 更新tmp_val
            tmp_val=Stack.top();
            Stack.pop();
        }

        return res;  

    }
};
```
