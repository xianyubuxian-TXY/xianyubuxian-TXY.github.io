# 1.常用语法

## （1）创建章内链接
### <1> 定义锚点
```html
<a id="describtion"></a>
# 标题1
```

### <2> 生成链接
```markdown
[文字描述](#describtion)
```

## （2）章外链接（引用其它文章）
### <1>绝对路径链接
`[显示文字](链接URL路径)`

### <2>Hexo 内置标签（推荐，不会失效）
`{% post_link "文件路径" '显示文本' %}`

如：
```text
{% post_link "C++/C++特性/CPP模板基础语法" 'C++模板基础语法' %}
```
