一、nginx简介

1.什么是nginx

  • 高性能的 HTTP 服务器 和 反向代理服务器
  • 支持 TCP/UDP 代理（stream 模块）
  • 特点：高并发、低内存、热部署、高可用

2.核心功能

（1）Nginx的反向代理

  • 客户端对目标服务器是无感知的，客户端将请求发送到反向代理服务器，反向代理服务器再将请求发送到目标服务器，获取响应数据后返回给客户端 ——> 只暴露了反向代理服务器的IP地址，而隐藏了真实服务器的IP地址

（2）Nginx的负载均衡

  • 在“Nginx的反向代理”我们知道：客户端请求先发送到反向代理服务器，在由反向代理服务器发送到目标服务器
  • 而负载均衡就是让反向代理服务器通过指定的负载均衡策略，将请求合理的分发到不同目标服务器上，避免某个目标服务器接收过多请求而导致超载，而其它服务器却因为请求太少而利用率太低
